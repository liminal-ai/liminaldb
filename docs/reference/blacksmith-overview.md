# Blacksmith Overview

> Source: https://docs.blacksmith.sh/introduction/why-blacksmith

Blacksmith is a drop-in replacement for GitHub runners that offers faster execution, better observability, and lower costs.

## Features

### Performance

- **2x faster runners** - Bare metal gaming CPUs with highest single-core performance
- **4x faster cache downloads** - Cache artifacts in same data center
- **40x faster Docker builds** - Persist Docker layers on NVMe drives
- **Docker pull cache** - Cache public Docker images

### Observability

- **Run History** - Search, filter, debug past CI runs
- **Logs** - Search and filter across entire CI pipeline
- **SSH Access** - Debug running jobs, inspect VM state
- **Test Analytics** - Identify and fix test failures quickly
- **CI Analytics** - Monitor performance and costs across team
